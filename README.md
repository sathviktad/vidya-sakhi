# 🎓 Vidya Sakhi – AI-Powered Learning Companion for Students

**Vidya Sakhi** is a smart, multilingual, AI-powered Streamlit web application tailored to help school students from **classes 3 to 12**. It offers:

- 📚 AI study assistant with voice and language support  
- 🧠 AI-generated subject quizzes (Math, Science, GK, English, Social, Regional, Hindi, etc.)  
- 🎙️ Speech output for accessibility  
- 🔤 Multilingual support (Hindi, Telugu, Tamil, etc.)  
- 🧑‍🏫 Role-based views for Students, Teachers, and Admins  
- 📊 Visual performance analytics  
- 🗂️ Study resources upload, search, and access  

---

## 💡 Features

### 👩‍🎓 Student View
- 💬 Chatbot in multiple languages with voice playback
- 📝 AI-generated MCQ quizzes
- 📚 Subject-wise study materials
- 📊 Progress analytics and feedback
- 🌐 Language switcher

### 👩‍🏫 Teacher View
- 📝 Add quizzes by class, subject, topic
- 📚 Upload study resources (PDF, links, etc.)
- 📊 See quiz participation & student performance

### 🛡️ Admin View
- 🔐 Secure login
- 👥 Manage users (students, teachers)
- 🗂️ Monitor overall data and analytics

---

## ⚙️ Tech Stack

- **Frontend:** Streamlit + HTML/CSS + JS (for voice)
- **Backend:** Python + DeepSeek AI (via Groq)
- **Translation:** Translatepy and Googletrans
- **Text-to-Speech:** Google Text-to-Speech (gTTs)
- **Hosting:** Streamlit Cloud (or localhost)
- **Roles:** Student, Teacher, Admin

---

## 🔧 How to Run Locally

```bash
git clone https://github.com/your-username/vidya-sakhi.git
cd vidya-sakhi
pip install -r requirements.txt
streamlit run app.py
```
